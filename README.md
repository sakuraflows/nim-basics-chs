### 使用教程
windows下推荐使用wsl2构建本文档
<br/>
依次运行如下指令，安装依赖
```
sudo apt install make ruby-full  安装make和ruby
sudo gem install asciidoctor     安装asciidoctor
```
<br/><br/>
构建文档
`make`
